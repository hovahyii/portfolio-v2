How to Get Pro Version in Github

4 February 2020 Hovah Yii Tutorials

If you are currently studying Software Engineering, Computer Science or ICT, one of the largest version control you should know and learn is Github. Github today has about 40 million users and more than 100 millions repositories, making it the largest host of source code in the world.

If you want to learn more about Github, please wait for my next articles about it.

So how to get Pro version of github?

First, go to https://education.github.com/pack.

Second, click 'Get the Pack'. You will be directed to this page.

Click 'Github Student Developer Pack'. Since I had already signed up the PRO package, I can't show you what the registration form looks like.

You will be asked to fill in a form. If you are using personal email, remember to include your student/school webmail, bind your school/student webmail to your personal webmail. If not, you will not be approved for PRO version. PRO version requires your student/school webmail.

After you filled the form and sent, wait for few days, the application usually takes 4 to 5 days to process. Remember to constantly check your email.

Once approved, you will see the PRO badge in 'Your Profile'.

Congratulation, you are now accessible to unlimited commit and respositories and free stuffs that worth $45,000 USD.
